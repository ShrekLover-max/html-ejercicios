La Web 1.0, 2.0 y Web 3.0

Abstract

En este documento se habla el origen de la web 1.0 comparándolos con términos similares como el internet, las www y la web estática. También se habla de las aplicaciones de la web 2.0 y de sus diferencias con la web 1.0. (La web 2.0 es la web que tiene mas avances y es mas uso) Y por último se hace mención de la web 3.0 y todo lo que trajo consigo. De Facebook, como funciona y su relacion con el Web 3.0
